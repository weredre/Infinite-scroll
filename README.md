# Infinite-scroll

Simple overview of use/purpose.

## Description

A mock Pinterest style website using Infinite Scroll and Masonry styling.

Used the Infinite Scroll API for Front-End
Used JSONBIN.io for Back-End

## Getting Started

### Dependencies

No Dependencies

### Executing program

```
npm start
```

## Help


```
npm install
```
